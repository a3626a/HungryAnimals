Introduction

'food_preferences/item/modid#name.json' contains list of ingredients which provides "nutrient" and "stomach". The animal eats fallen the items, is temped by the items, can be fed by the items.

Grammar

"food_preferences/item/modid#animal.json" : 
    [
        {
            "item": ingredient,
            "nutrient": double,
            "stomach": double
        },
        ...
    ]

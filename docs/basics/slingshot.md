# Slingshot

## Introduction

`slingshot.json` contains list of 'Ingredient' which slingshots can fire.

## Grammar

{% code title="slingshot.json" %}
```text
{
  "ammos" : [
    ingredient,
    ingredient,
    ...
  ],
  "damage" : double
}
```
